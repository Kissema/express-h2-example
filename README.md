# express-h2-example
Creating a express server with http2
## The three servers running examples
- Server with HTTP 1.1
    ```bash
    npm run http1
    ```
- Server with HTTP 2
    ```bash
    npm run http2
    ```
- Server with HTTP 1.1 and HTTP 2
    ```bash
    npm start
    ```

Congrants to [express-h2](https://github.com/ksoliddev/express-h2)
